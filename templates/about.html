What is Flog really about?

  • FLog is a dynamic Web Application built using Flask, a Python framework

  • The beautiful templates have been built upon Jinja2 templating engine of Flask

  • All user details are securely stored on a MySQL database

  • User passwords are hashed and stored to ensure privacy

  • Minimal styling for the Web App is done using Bootstrap

  • CKEditor is used for styling the body of blog-posts
